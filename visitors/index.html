The map below shows sample data. To record real visitors, connect a free Firebase database:

  1. Go to console.firebase.google.com and sign in with your Google account.
  2. Click Add project → enter any project name → Continue → Create project.
  3. In the left sidebar: Build → Realtime Database → Create database → Start in test mode → Enable.
  4. Copy the database URL shown at the top — it looks like https://your-project-default-rtdb.firebaseio.com
  5. Open visitors/index.html, find const FB_URL = 'YOUR_FIREBASE_URL' near line 310, and paste your URL.
