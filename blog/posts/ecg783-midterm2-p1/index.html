ECG 783 Midterm #2 — Problem 1: Adaptive System Identification via LMS, SER, and RLS

Course: ECG 783  ·  Student: Chang Wang (2002721058)  ·  Instructor: Peter Stubberud  ·  04-27-2026

1. Problem Statement

Model an unknown single-input single-output (SISO) system using a 10-weight adaptive linear combiner (10-tap FIR filter). The "unknown" system is the second-order IIR plant given by the difference equation:

$$y(n) = \tfrac{1}{2}y(n-1) - \tfrac{1}{4}y(n-2) + \tfrac{1}{8}y(n-3) + x(n) + \tfrac{3}{2}x(n-1) + \tfrac{1}{4}x(n-2).$$

Three adaptive algorithms are applied in parts (a), (b), and (c):

  • (a) LMS — Least Mean Squares
  • (b) SER — Stochastic Expectation Rescaling (inverse-correlation LMS)
  • (c) RLS — Recursive Least Squares

Each algorithm must achieve misadjustment $M \le 1\%$. Results include: final weight vectors, weight-trajectory plots, average learning curves with $T_\text{MSE}$ estimates, and frequency-response comparisons.

2. Common Problem Setup

2.1 Unknown System — Transfer Function

Taking the $z$-transform and rearranging:

$$H(z) = \frac{Y(z)}{X(z)} = \frac{1 + \tfrac{3}{2}z^{-1} + \tfrac{1}{4}z^{-2}} {1 - \tfrac{1}{2}z^{-1} + \tfrac{1}{4}z^{-2} - \tfrac{1}{8}z^{-3}}.$$

2.2 FIR Adaptive Model

The adaptive linear combiner is implemented as a 10-tap FIR filter. Its input vector, output, and error signal are:

$$\mathbf{x}(n) = \bigl[x(n),\, x(n-1),\, \ldots,\, x(n-9)\bigr]^T,$$ $$\hat{y}(n) = \mathbf{w}^T(n)\,\mathbf{x}(n), \qquad e(n) = d(n) - \hat{y}(n),$$

where $d(n) = y(n)$ is the output of the unknown IIR plant.

3. Wiener Solution and Optimal 10-Weight Approximation

3.1 Mean-Square Error and Wiener–Hopf Equation

The MSE surface is:

$$J(\mathbf{w}) = E\!\left[e^2(n)\right] = E\!\left[(d(n) - \mathbf{w}^T\mathbf{x}(n))^2\right].$$

The unique global minimum is the Wiener solution:

$$\mathbf{w}_\text{opt} = R^{-1}\mathbf{p},$$ $$R = E\!\left[\mathbf{x}(n)\mathbf{x}^T(n)\right], \qquad \mathbf{p} = E\!\left[d(n)\mathbf{x}(n)\right].$$

3.2 White-Input Simplification

Because the input $x(n)$ is zero-mean white noise with unit variance, the autocorrelation matrix is simply the identity:

$$R = I_{10} \implies \mathbf{w}_\text{opt} = \mathbf{p}.$$

For white-noise system identification, $\mathbf{p}$ equals the first ten samples of the plant's impulse response $h[n]$. The theoretical optimal 10-tap weights are therefore:

$$\mathbf{w}_\text{opt} = \bigl[1,\; 2,\; 1,\; 0.125,\; 0.0625,\; 0.125,\; 0.0625,\; 0.0078125,\; 0.00390625,\; 0.0078125\bigr]^T.$$
Minimum MSE due to IIR tail truncation:
$J_\min = 1.585717 \times 10^{-5}$ (energy of the impulse-response tail beyond tap 9).

Part (a) — LMS

6. LMS Results

6.1 Algorithm and Parameter Design

The LMS weight update rule is:

$$\mathbf{w}(n+1) = \mathbf{w}(n) + 2\mu\, e(n)\,\mathbf{x}(n).$$

The approximate misadjustment for this form is $M \approx \mu\,\text{tr}(R)$. Since $R = I_{10}$, we have $\text{tr}(R) = 10$, so $M \approx 10\mu$. Choosing $\mu = 0.0008$ gives:

$$M = 10 \times 0.0008 = 0.008 = 0.8\% \le 1\%. \checkmark$$

6.2 Theoretical Learning-Curve Time Constant

With $R = I_{10}$, all eigenvalues $\lambda_k = 1$. The LMS learning-curve time constant is:

$$T_\text{MSE} = \frac{1}{4\mu\lambda_\min} = \frac{1}{4 \times 0.0008 \times 1} = 312.5 \text{ iterations}.$$

6.3 Weight Trajectories

The average LMS weight trajectories (over 100 trials) show stable convergence. The three dominant weights $w_0 \to 1$, $w_1 \to 2$, and $w_2 \to 1$ converge smoothly. The smaller weights also settle near their theoretical values with minor stochastic fluctuations from gradient noise.

6.4 Average Learning Curve

The average of 100 LMS learning curves was computed. An exponential fit to the transient gives an estimated time constant of $T_\text{MSE,est} = 315.787$ iterations — differing from the theoretical 312.5 by only $\approx 1.05\%$, confirming the convergence analysis. The steady-state MSE settles at $J_\text{ss} = 1.628 \times 10^{-5}$, close to $J_\min = 1.586 \times 10^{-5}$.

6.5 Frequency Response Comparison

The 10-weight LMS filter frequency response nearly overlaps that of the unknown IIR system in both magnitude and phase across the full normalized frequency range $[0,\,\pi]$. The small remaining deviation is expected: a 10-tap FIR can only approximate an infinite-length IIR impulse response.

Part (b) — SER

7. SER Results

7.1 Algorithm and Parameter Design

SER scales the update direction using an exponentially windowed inverse-correlation estimate $Q^{-1}(n)$. The update rule is:

$$\mathbf{w}(n+1) = \mathbf{w}(n) + 2\mu\, Q^{-1}(n)\, e(n)\,\mathbf{x}(n),$$ $$Q(n) = \alpha\, Q(n-1) + (1-\alpha)\,\mathbf{x}(n)\mathbf{x}^T(n), \quad Q^{-1}(0) = 10I.$$

Parameters: memory factor $\alpha = 0.995$, step size $\mu = 0.0008$. For white input ($R = I_{10}$, $\lambda_\text{av} = 1$):

$$M \approx 10\mu = 0.8\% \le 1\%, \qquad T_\text{MSE} = \frac{1}{4\mu\lambda_\text{av}} = 312.5 \text{ iterations}.$$

7.2 Design Note

Because the input is white and $R = I_{10}$, the advantage of SER over LMS is minimal here — the correlation matrix is already diagonal and well-conditioned, so rescaling provides no convergence benefit. Both algorithms exhibit similar convergence speed and misadjustment.

7.3 Weight Trajectories

SER weight trajectories converge to the same Wiener solution as LMS. The smaller coefficients show small transient fluctuations but settle to the correct values.

7.4 Average Learning Curve

The estimated SER time constant is $T_\text{MSE,est} = 320.378$ iterations, differing from the theoretical 312.5 by $\approx 2.52\%$. The steady-state MSE is $J_\text{ss} = 1.618 \times 10^{-5}$, close to $J_\min$.

7.5 Frequency Response Comparison

The SER frequency response closely matches the unknown system in both magnitude and phase, confirming the final SER weights provide an accurate 10-tap FIR approximation.

Part (c) — RLS

8. RLS Results

8.1 Algorithm and Parameter Design

RLS uses an exact recursive estimate of the inverse correlation matrix $P(n) = R^{-1}(n)$. The forgetting factor $\lambda = 1$ is appropriate since the plant is time-invariant. Initialization with $\delta = 0.01$ gives $P(0) = \delta^{-1}I = 100I$, providing a large initial adaptation gain for very rapid convergence.

The RLS recursion is:

$$\mathbf{k}(n) = \frac{P(n-1)\,\mathbf{x}(n)} {\lambda + \mathbf{x}^T(n)\,P(n-1)\,\mathbf{x}(n)},$$ $$\mathbf{w}(n+1) = \mathbf{w}(n) + \mathbf{k}(n)\,e(n),$$ $$P(n) = \lambda^{-1}\bigl[P(n-1) - \mathbf{k}(n)\,\mathbf{x}^T(n)\,P(n-1)\bigr].$$

8.2 Weight Trajectories

The RLS weight trajectories show that nearly all convergence occurs within the first few tens of iterations — dramatically faster than the thousands of iterations required by LMS and SER.

8.3 Average Learning Curve and $T_\text{MSE}$ Estimation

Since RLS does not use a fixed scalar step size, the LMS-style theoretical $T_\text{MSE}$ does not apply. Instead, $T_\text{MSE}$ is estimated directly from the averaged learning curve using a log-linear fit to the transient portion:

$$T_\text{MSE,est} = 1.650 \text{ iterations.}$$

This indicates extremely fast convergence. The steady-state MSE is $J_\text{ss} = 1.611 \times 10^{-5}$, the closest to $J_\min$ among all three algorithms.

8.4 Frequency Response Comparison

The RLS frequency response is almost identical to the unknown system over the full normalized frequency range, confirming that the RLS-derived 10-weight FIR filter accurately captures the dominant plant dynamics.

Conclusions:
  • All three algorithms converge to the same mean-square-optimal 10-weight FIR approximation of the IIR plant, with final weights closely matching the theoretical Wiener solution $\mathbf{w}_\text{opt}$.
  • LMS and SER (both $\mu = 0.0008$) achieve $M = 0.8\% \le 1\%$ and converge with time constants of ~315–320 iterations, agreeing well with the theoretical 312.5 iterations derived from $T_\text{MSE} = 1/(4\mu\lambda_\min)$.
  • RLS converges in approximately 1.65 iterations — roughly 200× faster than LMS/SER — because it directly inverts the sample correlation matrix rather than following a noisy gradient. It also achieves the smallest coefficient errors and the steady-state MSE closest to $J_\min$.
  • The frequency-response plots for all three algorithms closely match the unknown IIR system in both magnitude and phase, validating that a 10-tap FIR provides an accurate mean-square approximation of the given IIR plant.
