ECG 783 Midterm #2 — Problem 4: FFNN Two-Input Nonlinear System Modeling

2026-04-27  ·  ECE · Adaptive Signal Processing

Problem 4

1. Problem Setup

The objective is to model an unknown two-input single-output nonlinear system using a two-layer feedforward neural network (FFNN). The "unknown" system is defined as

$$y(k) = \cos\!\bigl[\pi x_1(k)\bigr]\cos\!\bigl[\pi x_2(k)\bigr] + 0.25\,x_1^2(k)$$

over the input domain $-1 \le x_1(k) \le 1$ and $-1 \le x_2(k) \le 1$. The FFNN must have at least 6 inputs and at least 50 hidden neurons, and must achieve a maximum absolute approximation error below $0.15$.

Part (a)

2. Unknown System Surface

The system was evaluated on a uniform grid with step size $0.1$ over $[-1,\,1] \times [-1,\,1]$, generating the MATLAB surface via:

step = 0.1; x1 = -1:step:1; x2 = -1:step:1; [X1,X2] = meshgrid(x1,x2); y = cos(pi*X1).*cos(pi*X2) + 0.25*X1.^2;

The product of the two cosine terms makes the surface highly nonlinear, with sharp peaks and valleys. The additional quadratic term $0.25\,x_1^2(k)$ shifts the surface upward symmetrically as $|x_1|$ increases.

Part (b)

3. FFNN Input Design

Although the physical system has only two inputs, the problem requires the network to use at least six inputs. The six-dimensional input vector is constructed from the previous, current, and next values of both input variables:

$$X = \begin{bmatrix} x_1(k_1 - 1) \\ x_1(k_1) \\ x_1(k_1 + 1) \\ x_2(k_2 - 1) \\ x_2(k_2) \\ x_2(k_2 + 1) \end{bmatrix} \in \mathbb{R}^6$$

The target output remains the true system value at the center grid point:

$$T = f\bigl(x_1(k_1),\; x_2(k_2)\bigr).$$

This neighbourhood context gives the network local gradient information about the surface curvature without changing the fundamental mapping being learned.

Training Dataset

A finer grid with step size $0.05$ was used for training, producing 1681 samples:

  • Input matrix: $P \in \mathbb{R}^{6 \times 1681}$
  • Target matrix: $T \in \mathbb{R}^{1 \times 1681}$

4. Network Structure and Training Algorithm

The two-layer FFNN model is

$$\hat{y}(k) = W_2\,\phi(W_1 X + b_1) + b_2$$

where $\phi(\cdot)$ is the hidden-layer activation and $\hat{y}(k)$ is the network output. The final architecture was:

6 inputs  →  60 hidden neurons (tansig)  →  1 output (purelin)
  • Hidden layer: 60 neurons with tansig activation — chosen to exceed the 50-neuron minimum and provide ample representational capacity for the strongly nonlinear $\cos \times \cos$ surface.
  • Output layer: 1 neuron with purelin (linear) activation, appropriate for a regression target that ranges continuously over $(-1, 1.25)$.
  • Training algorithm: Levenberg–Marquardt (trainlm), which combines the speed of Gauss–Newton with the robustness of gradient descent.
  • Performance metric: Mean squared error (MSE).

The approximation error at each grid point is defined as

$$e(k) = y(k) - \hat{y}(k),$$

and the requirement is $\max|e(k)| < 0.15$.

5. Results

Figure 3 shows the pointwise error $e(k) = y(k) - \hat{y}(k)$ across the full grid. The errors are uniformly small and remain on the order of $10^{-4}$, with the largest deviation of $0.000804$ occurring near the boundary of the domain.

6. Discussion

Why 6 Inputs?

Using the previous, current, and next values of $x_1$ and $x_2$ provides the network with local finite-difference information. This effectively supplies approximate first-order context about how the surface changes across the grid, helping the network learn the curvature of the $\cos \times \cos$ product without requiring a deeper architecture.

Why 60 Hidden Neurons?

The target surface $y(k) = \cos(\pi x_1)\cos(\pi x_2) + 0.25\,x_1^2$ is strongly nonlinear: the $\cos \times \cos$ term alone creates four lobes with alternating signs over $[-1,1]^2$. A single hidden layer with 60 tansig neurons provides sufficient capacity to partition this input space accurately. Fewer neurons (e.g., the minimum of 50) also achieved $\max|e| < 0.15$, but 60 neurons improved accuracy by roughly $10\times$ on the first training trial.

Levenberg–Marquardt vs. Standard Backprop

For small-to-medium networks like this one ($6 \times 60 + 60 \times 1 = 420$ weights), Levenberg–Marquardt converges in far fewer iterations than vanilla gradient descent by approximating the Hessian. The best network was obtained on the very first training trial, indicating that LM found a deep minimum efficiently from the random initialisation.

7. Conclusion

A two-layer FFNN (6 inputs → 60 hidden neurons → 1 output) trained with Levenberg–Marquardt successfully modeled the nonlinear system $y(k) = \cos(\pi x_1)\cos(\pi x_2) + 0.25\,x_1^2$ over $[-1,1]^2$. Key results:
