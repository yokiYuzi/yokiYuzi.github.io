ECG 783 HW1 — Wiener Solution for the Adaptive Linear Combiner

Course: ECG 783  ·  Student: Chang Wang (2002721058)  ·  Instructor: Peter Stubberud  ·  02-22-2026

1. Objective

Compute the Wiener (minimum-MSE) solution for an Adaptive Linear Combiner (ALC) that models the desired signal $d(k)$ from the input $x(k)$. For each filter length $L+1$, we estimate the autocorrelation matrix $R$ and cross-correlation vector $P$, solve the Wiener–Hopf equations

$$R\,W = P,$$

and verify the solution via the residual $RW - P$. Time-domain performance is evaluated by comparing $y(k)$ to $d(k)$ and plotting $e^2(k)$; the filter's frequency response $H(e^{j\omega})$ is also analyzed.

2. ALC Model and Wiener Formulation

The ALC is an FIR model with $L+1$ coefficients:

$$y(k) = \sum_{n=0}^{L} w_n\, x(k-n) = W^T x_k,$$ $$x_k = [x(k)\ \ x(k-1)\ \ \cdots\ \ x(k-L)]^T, \quad W = [w_0\ \ w_1\ \ \cdots\ \ w_L]^T.$$

The MSE cost function is $J(W)=\mathbb{E}[(d(k)-W^Tx_k)^2]$. Setting $\nabla J=0$:

$$R = \mathbb{E}[x_k x_k^T], \quad P = \mathbb{E}[d(k)\,x_k].$$

Finite-sample biased estimates for lag $\ell=0,\ldots,L$:

$$\hat{r}_{xx}(\ell) = \frac{1}{N}\sum_{k=\ell}^{N-1} x(k)\,x(k-\ell), \qquad \hat{r}_{dx}(\ell) = \frac{1}{N}\sum_{k=\ell}^{N-1} d(k)\,x(k-\ell).$$ $$\hat{R} = \operatorname{Toeplitz}\!\bigl([\hat{r}_{xx}(0),\ldots,\hat{r}_{xx}(L)]^T\bigr), \quad \hat{P} = [\hat{r}_{dx}(0),\ldots,\hat{r}_{dx}(L)]^T, \quad \hat{W}=\hat{R}\backslash\hat{P}.$$

Case 1: Sinusoidal input — two taps achieve near-perfect modeling. Extra taps carry negligible weight; the filter acts as a gain-and-phase shifter at $\omega_0$.

Case 2: Square-wave input — rich harmonic content demands more taps. Two taps cannot selectively suppress $3\omega_0$ ($-0.91\,\text{dB}$), yielding MSE $\approx0.537$. Ten taps notch $3\omega_0$ by $-22.6\,\text{dB}$, reducing MSE by $\approx84\%$.

Conclusion: Filter length matters only when the input contains spectral components that must be suppressed. For single-tone inputs the minimum sufficient length is determined by the amplitude-phase constraint at $\omega_0$ alone.
