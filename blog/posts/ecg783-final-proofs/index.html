ECG 783 Final Exam Review — Five Proof Problems

2026-05-13  ·  ECE · Adaptive Signal Processing

Five derivation-and-proof problems at final-exam level, covering Ch1–8: the Wiener–Hopf equation and orthogonality principle, steepest-descent principal-coordinate analysis and learning curve, finite-difference gradient estimation performance/variance trade-off, regularized LMS with an arbitrary reference vector (generalised leaky LMS), and the $z$-domain performance function together with the SER matrix-inversion-lemma recursion. Every result is derived from first principles with no formula substituted without justification.

Problem 1

Wiener Solution, Orthogonality & Shifted Performance Surface

Setup

Consider the adaptive linear combiner $y(n)=W^T X_n$, $e(n)=d(n)-W^T X_n$, with $R=E[X_n X_n^T]$, $P=E[d(n)X_n]$, $\sigma_d^2=E[d^2(n)]$. Assume $R$ is non-singular.

Proof — $J(W) = \sigma_d^2 - 2P^T W + W^T R W$

Starting from $e^2(n)=[d(n)-W^T X_n]^2$, expand:

$$e^2(n) = d^2(n) - 2\,d(n)\,W^T X_n + W^T X_n X_n^T W.$$

Take the expectation. Since $W$ is a deterministic vector it factors out of $E[\cdot]$:

$$E[d(n)\,W^T X_n] = W^T E[d(n)X_n] = W^T P = P^T W,$$ $$E[W^T X_n X_n^T W] = W^T E[X_n X_n^T] W = W^T R W.$$

Therefore:

$$\boxed{J(W) = \sigma_d^2 - 2P^T W + W^T R W.}$$
□
Proof — $W^* = R^{-1}P$

Differentiate $J(W)$ with respect to $W$:

$$\nabla J(W) = -2P + 2RW.$$

Set $\nabla J(W^*) = 0$:

$$2RW^* = 2P \implies RW^* = P.$$

Since $R$ is non-singular:

$$\boxed{W^* = R^{-1}P.}$$
□
Proof — Shifted form $J(W) = J_{\min} + (W-W^*)^T R\,(W-W^*)$

Let $V = W - W^*$, so $W = W^* + V$. Substitute into $J$:

$$J = \sigma_d^2 - 2P^T(W^*+V) + (W^*+V)^T R(W^*+V).$$

Expanding the quadratic:

$$(W^*+V)^T R(W^*+V) = W^{*T}RW^* + 2V^T RW^* + V^T RV.$$

Because $RW^* = P$, the cross-terms cancel exactly: $-2P^TV + 2V^T RW^* = -2P^TV + 2V^T P = 0$.

What remains is:

$$J = \underbrace{\sigma_d^2 - 2P^T W^* + W^{*T}RW^*}_{J_{\min}} + V^T RV.$$

Since $RW^* = P$ implies $W^{*T}RW^* = W^{*T}P = P^T W^*$, we confirm $J_{\min} = \sigma_d^2 - P^T W^*$. Hence:

$$\boxed{J(W) = J_{\min} + (W-W^*)^T R\,(W-W^*).}$$
□
Proof — Orthogonality $E[e^*(n)X_n] = 0$

At the optimum, $e^*(n) = d(n) - W^{*T}X_n$, so:

$$E[e^*(n)X_n] = E[d(n)X_n] - E[X_n X_n^T W^*] = P - RW^*.$$

Since $RW^* = P$:

$$\boxed{E[e^*(n)X_n] = 0.}$$

This is the Wiener–Hopf orthogonality principle: at the Wiener optimum the error is orthogonal to every component of the input vector.

□
Problem 2

Steepest Descent: Principal-Coordinate Recursion, Stability & Learning Curve

Setup

Performance surface $J(W) = J_{\min} + (W-W^*)^T R(W-W^*)$, $R = Q\Lambda Q^T$ (eigen-decomposition, $Q$ orthogonal, $\Lambda = \operatorname{diag}(\lambda_0,\ldots,\lambda_L)$). Steepest descent: $W_{k+1} = W_k - \mu\nabla J(W_k)$. Define $V_k = W_k - W^*$ and $V_k = Q V_k'$ (principal coordinates).

Proof — Principal-coordinate recursion $V'_{k+1} = (I-2\mu\Lambda)V'_k$

From the shifted form, $\nabla J(W) = 2R(W-W^*) = 2RV_k$. The update becomes:

$$W_{k+1} - W^* = W_k - W^* - 2\mu R(W_k - W^*) \implies V_{k+1} = (I-2\mu R)V_k.$$

Substitute $V_k = QV_k'$:

$$QV'_{k+1} = (I-2\mu R)QV'_k.$$

Left-multiply by $Q^T$ and use $Q^T Q = I$, $Q^T R Q = \Lambda$:

$$\boxed{V'_{k+1} = (I - 2\mu\Lambda)V'_k,}$$

so $v_i'(k) = (1-2\mu\lambda_i)^k\,v_i'(0)$ for each mode $i$ independently.

□
Proof — Stability condition $0 < \mu < 1/\lambda_{\max}$

Mode $i$ converges if and only if $|1-2\mu\lambda_i|<1$, i.e. $-1 < 1-2\mu\lambda_i < 1$. The right inequality gives $\mu > 0$; the left gives $\mu < 1/\lambda_i$. For all modes to converge simultaneously:

$$\boxed{0 < \mu < \frac{1}{\lambda_{\max}}.}$$
□
Proof — Learning curve $J(k) = J_{\min} + \sum_i \lambda_i [v_i'(0)]^2 (1-2\mu\lambda_i)^{2k}$

In principal coordinates $V_k = QV_k'$, so:

$$V_k^T R V_k = V_k^{\prime T} Q^T R Q\, V_k' = V_k^{\prime T}\Lambda V_k' = \sum_{i=0}^{L}\lambda_i[v_i'(k)]^2.$$

Substituting $v_i'(k) = (1-2\mu\lambda_i)^k v_i'(0)$:

$$\boxed{J(k) = J_{\min} + \sum_{i=0}^{L}\lambda_i\,[v_i'(0)]^2\,(1-2\mu\lambda_i)^{2k}.}$$

Each term decays with its own time constant $\tau_{\mathrm{MSE},i} = -1/\ln(1-2\mu\lambda_i)^2 \approx 1/(4\mu\lambda_i)$ for small $\mu\lambda_i$. The overall learning curve is a sum of decaying exponentials, with the slowest mode ($\lambda_{\min}$) dominating at large $k$.

□
Problem 3

Finite-Difference Gradient Estimation: Perturbation & Variance

Setup

In principal coordinates, $J(V') = J_{\min}+\sum_i\lambda_i(v_i')^2$. Central-difference gradient estimator for direction $i$: $\hat{g}_i = [\hat J(V'+se_i)-\hat J(V'-se_i)]/(2s)$, with $\operatorname{var}[\hat J]=\sigma_J^2$ at both perturbed points (independent).

Proof — Exact gradient $g_i = 2\lambda_i v_i'$

Only the $i$-th term changes under perturbation $\pm s e_i$:

$$J(V'\!+\!se_i)-J(V'\!-\!se_i) = \lambda_i\bigl[(v_i'\!+\!s)^2-(v_i'\!-\!s)^2\bigr] = \lambda_i\cdot 4v_i' s.$$ $$\boxed{g_i = \frac{4\lambda_i v_i' s}{2s} = 2\lambda_i v_i'.}$$
□
Proof — Performance perturbation $\gamma_i = \lambda_i s^2$

The two-point average approximates the true performance at $V'$ but evaluates at $\pm s$:

$$\tilde J(V') = \tfrac{1}{2}[J(V'\!+\!se_i)+J(V'\!-\!se_i)] = J(V') + \lambda_i s^2.$$ $$\boxed{\gamma_i = \tilde J(V') - J(V') = \lambda_i s^2.}$$

Normalised perturbation: $P_i = \lambda_i s^2/J_{\min}$. Averaged over all $L+1$ directions:

$$\boxed{P = \frac{1}{L+1}\sum_{i=0}^{L}P_i = \frac{s^2}{(L+1)J_{\min}}\operatorname{tr}(R) = \frac{s^2\,\lambda_{\mathrm{ave}}}{J_{\min}}.}$$
□
Proof — Gradient estimate variance $\operatorname{var}[\hat{g}_i] = \sigma_J^2/(2s^2)$

Since the two measurements are independent:

$$\operatorname{var}[\hat{g}_i] = \operatorname{var}\!\left[\frac{\hat J_+ - \hat J_-}{2s}\right] = \frac{\operatorname{var}[\hat J_+]+\operatorname{var}[\hat J_-]}{4s^2} = \frac{2\sigma_J^2}{4s^2}.$$ $$\boxed{\operatorname{var}[\hat{g}_i] = \frac{\sigma_J^2}{2s^2}.}$$
Fundamental trade-off: Performance penalty $\gamma_i = \lambda_i s^2$ grows with $s$; gradient variance $\sigma_J^2/(2s^2)$ grows as $s\to 0$. The perturbation tolerance $P$ is therefore the primary design constraint for $s$.
□
Problem 4

Regularized LMS with Reference Vector — Generalised Leaky LMS

Setup

Adaptive linear combiner with regularized cost $J(W_n) = E[e^2(n)] + \alpha\|W_n - W_0\|^2$, where $\alpha>0$ and $W_0$ is a given reference vector.

Proof — Steepest-descent update

From Problem 1, $E[e^2(n)] = \sigma_d^2 - 2P^TW + W^TRW$. The regularisation term expands as

$$\alpha\|W-W_0\|^2 = \alpha(W-W_0)^T(W-W_0).$$

Differentiate the full cost:

$$\nabla J(W) = -2P + 2RW + 2\alpha(W-W_0) = 2(R+\alpha I)W - 2(P+\alpha W_0).$$

Steepest descent $W_{n+1} = W_n - \mu\nabla J(W_n)$ gives:

$$\boxed{W_{n+1} = [I - 2\mu(R+\alpha I)]\,W_n + 2\mu(P+\alpha W_0).}$$
□
Proof — LMS form

Replace $E[\cdot]$ by the instantaneous sample. The instantaneous gradient of $\hat J(n) = e^2(n)+\alpha\|W_n-W_0\|^2$ is

$$\nabla\hat J(n) = -2e(n)X_n + 2\alpha(W_n-W_0).$$

LMS update $W_{n+1} = W_n - \mu\nabla\hat J(n)$:

$$\boxed{W_{n+1} = (1-2\mu\alpha)W_n + 2\mu e(n)X_n + 2\mu\alpha W_0.}$$

When $W_0 = 0$ this reduces to standard leaky LMS $W_{n+1} = (1-2\mu\alpha)W_n + 2\mu e(n)X_n$.

□
Proof — Stability range $0 < \mu < 1/(\lambda_{\max}+\alpha)$

The steepest-descent propagation matrix is $A = I - 2\mu(R+\alpha I)$. Its eigenvalues are $1-2\mu(\lambda_i+\alpha)$. Convergence requires $|1-2\mu(\lambda_i+\alpha)|<1$ for all $i$, giving $\mu < 1/(\lambda_i+\alpha)$. The tightest constraint is the largest eigenvalue:

$$\boxed{0 < \mu < \frac{1}{\lambda_{\max}+\alpha}.}$$

Note: regularisation reduces the allowable step size relative to standard LMS, but it also makes the Hessian $R+\alpha I$ better conditioned.

□
Proof — Steady-state mean $\lim_{n\to\infty}E[W_n] = (R+\alpha I)^{-1}(P+\alpha W_0)$

Let $m_n = E[W_n]$. Under the standard independence assumption, $E[e(n)X_n] = P - Rm_n$. Taking the expectation of the LMS update:

$$m_{n+1} = (1-2\mu\alpha)m_n + 2\mu(P - Rm_n) + 2\mu\alpha W_0 = [I-2\mu(R+\alpha I)]\,m_n + 2\mu(P+\alpha W_0).$$

At steady state $m_\infty = [I-2\mu(R+\alpha I)]\,m_\infty + 2\mu(P+\alpha W_0)$, so

$$2\mu(R+\alpha I)\,m_\infty = 2\mu(P+\alpha W_0).$$ $$\boxed{\lim_{n\to\infty}E[W_n] = (R+\alpha I)^{-1}(P+\alpha W_0).}$$

When $W_0=0$: $m_\infty = (R+\alpha I)^{-1}P$ — the standard leaky-LMS biased solution. As $\alpha\to 0$: $m_\infty\to R^{-1}P = W^*$ — the unregularized Wiener solution. As $\alpha\to\infty$: $m_\infty\to W_0$ — the weight vector is pulled entirely to the prior.

□
Problem 5

$z$-Domain Performance Function & SER Inverse-Matrix Recursion

Setup

Adaptive filter with transfer function $H(z)$, stationary input $x(n)$ with auto-PSD $\Phi_{xx}(z)$, desired signal $d(n)$ with cross-PSD $\Phi_{dx}(z)$ relative to $x(n)$, error $e(n) = d(n)-y(n)$.

Proof — General $z$-domain MSE formula

Expand $J = E[e^2(n)] = E[d^2]-2E[dy]+E[y^2]$.

  • $E[d^2(n)] = \phi_{dd}(0)$.
  • For LTI filter $y(n)=h(n)*x(n)$, the output PSD is $\Phi_{yy}(z)=H(z)H(z^{-1})\Phi_{xx}(z)$, so $E[y^2(n)] = \phi_{yy}(0) = \mathcal{Z}^{-1}\{H(z)H(z^{-1})\Phi_{xx}(z)\}_{n=0}$.
  • The cross-correlation $\phi_{dy}(0) = E[d(n)y(n)]$. Since $y(n)=h(n)*x(n)$ and $\phi_{dx}(\ell)=E[d(n)x(n-\ell)]$, $\phi_{dy}(\tau) = \sum_k h(k)\phi_{dx}(\tau-k)$, i.e. $\Phi_{dy}(z)=H(z)\Phi_{dx}(z)$. Hence $E[d(n)y(n)] = \mathcal{Z}^{-1}\{H(z)\Phi_{dx}(z)\}_{n=0}$.

Combining:

$$\boxed{J = \phi_{dd}(0) + \mathcal{Z}^{-1}\!\left\{H(z)H(z^{-1})\Phi_{xx}(z) - 2H(z)\Phi_{dx}(z)\right\}_{n=0}.}$$
□
Proof — Reduction to FIR quadratic form

For $H(z)=\sum_{\ell=0}^L w_\ell z^{-\ell}$, the time-domain output is $y(n)=W^T X_n$.

Cross-term:

$$E[d(n)y(n)] = \sum_{\ell=0}^L w_\ell E[d(n)x(n-\ell)] = \sum_{\ell=0}^L w_\ell \phi_{dx}(\ell) = P^T W,$$

where $P_\ell = \phi_{dx}(\ell)$.

Output power:

$$E[y^2(n)] = \sum_{i,j} w_i w_j E[x(n-i)x(n-j)] = \sum_{i,j} w_i R_{ij} w_j = W^T R W,$$

where $R_{ij} = \phi_{xx}(i-j)$ (Toeplitz).

Therefore:

$$\boxed{J(W) = \phi_{dd}(0) - 2P^T W + W^T R W,}$$

consistent with the time-domain derivation in Problem 1. The $z$-domain framework is thus a strict generalisation valid for both FIR and IIR filters.

□
Proof — SER inverse-matrix recursion via Matrix Inversion Lemma

The exponentially-weighted correlation matrix satisfies

$$C_n = \alpha C_{n-1} + X_n X_n^T.$$

Apply the Sherman–Morrison–Woodbury identity $(A+uv^T)^{-1} = A^{-1} - A^{-1}u(1+v^T A^{-1}u)^{-1}v^T A^{-1}$ with $A = \alpha C_{n-1}$, $u = v = X_n$:

$$A^{-1} = \frac{1}{\alpha}C_{n-1}^{-1}, \qquad A^{-1}u = \frac{1}{\alpha}C_{n-1}^{-1}X_n.$$

Define $S_n = C_{n-1}^{-1}X_n$. Then:

$$1 + v^T A^{-1}u = 1 + \frac{1}{\alpha}X_n^T C_{n-1}^{-1}X_n = \frac{\alpha + X_n^T S_n}{\alpha}.$$

Its reciprocal is $\alpha/(\alpha + X_n^T S_n)$. Substituting:

$$C_n^{-1} = \frac{1}{\alpha}C_{n-1}^{-1} - \frac{1}{\alpha}S_n\cdot\frac{\alpha}{\alpha+X_n^T S_n} \cdot\frac{1}{\alpha}S_n^T.$$

Collecting terms:

$$\boxed{C_n^{-1} = \frac{1}{\alpha}\!\left[C_{n-1}^{-1} - \frac{S_n S_n^T}{\alpha + X_n^T S_n}\right], \qquad S_n = C_{n-1}^{-1}X_n.}$$

This requires only one matrix–vector product $S_n = C_{n-1}^{-1}X_n$ and one scalar division per time step — $O(L^2)$ instead of $O(L^3)$ for direct inversion. The SER/LMS-Newton update then follows as $W_{n+1} = W_n + 2\mu/(1-\alpha)\cdot C_n^{-1}e(n)X_n$.

□
