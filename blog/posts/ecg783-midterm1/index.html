ECG 783 Midterm #1 — Wiener Solution, Leaky LMS & Nonlinear Adaptive Filter

2026-04-20  ·  ECE · Adaptive Signal Processing

Problem 1

1. Wiener Solution & Steepest-Descent Convergence

Setup

Consider the 2-weight adaptive linear combiner (ALC) in Figure 1a. The input is $x(n) = f(n) + r(n)$, where $f(n)$ is the periodic triangle wave shown in Figure 1b, $r(n)$ is zero-mean white noise with variance $\sigma_r^2 = \tfrac{1}{2}$, and the desired signal is $d(n) = 6f(n-1)$.

Part (a) — Optimal Weight Vector $\mathbf{W}^*$

One period of $f(n)$ reads $\{0,\,-1,\,-2,\,-1,\,0,\,1,\,2,\,1\}$, giving:

$$E[f^2(n)] = \frac{0^2+1^2+2^2+1^2+0^2+1^2+2^2+1^2}{8} = \frac{12}{8} = \frac{3}{2}$$ $$E[f(n)f(n-1)] = 1$$

Because $r(n)$ is white and uncorrelated with $f(n)$:

$$E[x^2(n)] = E[f^2(n)] + E[r^2(n)] = \frac{3}{2} + \frac{1}{2} = 2$$ $$E[x(n)x(n-1)] = E[f(n)f(n-1)] = 1$$

The autocorrelation matrix and cross-correlation vector are therefore:

$$R = \begin{bmatrix}2 & 1 \\ 1 & 2\end{bmatrix}, \qquad P = \begin{bmatrix}E[d(n)x(n)]\\E[d(n)x(n-1)]\end{bmatrix} = \begin{bmatrix}6E[f(n-1)f(n)]\\6E[f^2(n-1)]\end{bmatrix} = \begin{bmatrix}6\\9\end{bmatrix}$$

Applying the Wiener–Hopf solution $\mathbf{W}^* = R^{-1}P$:

$$R^{-1} = \frac{1}{(2)(2)-(1)(1)}\begin{bmatrix}2&-1\\-1&2\end{bmatrix} = \frac{1}{3}\begin{bmatrix}2&-1\\-1&2\end{bmatrix}$$ $$\mathbf{W}^* = \frac{1}{3}\begin{bmatrix}2&-1\\-1&2\end{bmatrix} \begin{bmatrix}6\\9\end{bmatrix} = \frac{1}{3}\begin{bmatrix}3\\12\end{bmatrix} = \begin{bmatrix}1\\4\end{bmatrix}$$
Result: $\mathbf{W}^* = [1,\;4]^T$ for $X_n = [x(n),\,x(n-1)]^T$. (Equivalently $[4,\,1]^T$ if the regressor is ordered as $[x(n-1),\,x(n)]^T$.)

Part (b) — Steepest-Descent Convergence Range

The steepest-descent update is $\mathbf{W}_{n+1} = \mathbf{W}_n + 2\mu(P - R\mathbf{W}_n)$. Convergence requires $|1 - 2\mu\lambda_i| < 1$ for all eigenvalues $\lambda_i$ of $R$.

The eigenvalues of $R = \bigl[\begin{smallmatrix}2&1\\1&2\end{smallmatrix}\bigr]$ are $\lambda_1 = 3$ and $\lambda_2 = 1$. The binding constraint comes from $\lambda_\max = 3$:

$$\boxed{0 < \mu < \frac{1}{\lambda_{\max}} = \frac{1}{3}}$$
Problem 2

2. Leaky LMS Algorithm Derivation

Setup

For the adaptive system in Figure 2, the regularised cost function is:

$$J(\mathbf{W}_n) = E[e^2(n)] + \alpha E\!\left[\|\mathbf{W}_n\|^2\right] = E[e^2(n)] + \alpha E[\mathbf{W}_n^T\mathbf{W}_n], \quad \alpha > 0$$

Part (a) — Leaky LMS Update Rule

Using one sample $e(n)$ and $\mathbf{W}_n$ to estimate the gradient, the one-sample cost is $\hat{J}(\mathbf{W}_n) = e^2(n) + \alpha\mathbf{W}_n^T\mathbf{W}_n$. Taking the gradient:

$$\nabla\hat{J}(\mathbf{W}_n) = \frac{\partial e^2(n)}{\partial \mathbf{W}_n} + \frac{\partial(\alpha\mathbf{W}_n^T\mathbf{W}_n)}{\partial \mathbf{W}_n} = -2e(n)X_n + 2\alpha\mathbf{W}_n$$

Applying steepest descent $\mathbf{W}_{n+1} = \mathbf{W}_n - \mu\nabla\hat{J}$:

$$\mathbf{W}_{n+1} = \mathbf{W}_n - \mu[-2e(n)X_n + 2\alpha\mathbf{W}_n]$$ $$\boxed{\mathbf{W}_{n+1} = (1-2\mu\alpha)\mathbf{W}_n + 2\mu e(n)X_n}$$

The factor $(1-2\mu\alpha)$ is the "leak" — it shrinks the weight vector at each step, preventing unbounded growth.

Part (b) — Stability Range for $\mu$

Taking expectation and applying the LMS independence assumption:

$$E[\mathbf{W}_{n+1}] = [I - 2\mu(\alpha I + R)]\,E[\mathbf{W}_n] + 2\mu P$$

Stability requires every eigenvalue of $I - 2\mu(\alpha I + R)$ to have magnitude less than 1: $|1 - 2\mu(\alpha + \lambda_i)| < 1$. The binding constraint is at $\lambda_\max(R)$:

$$\boxed{0 < \mu < \frac{1}{\alpha + \lambda_{\max}(R)}}$$

Part (c) — Steady-State Solution

Let $m_n = E[\mathbf{W}_n]$ and $A = I - 2\mu(\alpha I + R)$. The recursion $m_{n+1} = Am_n + 2\mu P$ has the closed-form solution:

$$m_n = A^n m_0 + (I - A^n)(I-A)^{-1}(2\mu P)$$

Since $I - A = 2\mu(\alpha I + R)$, we have $(I-A)^{-1}(2\mu P) = (\alpha I + R)^{-1}P$. In the stable range $A^n \to 0$, so:

$$\boxed{\lim_{n\to\infty} E[\mathbf{W}_n] = (\alpha I + R)^{-1}P}$$

This is the biased Wiener solution — the regularisation term $\alpha I$ shifts the steady-state weights away from $R^{-1}P$, trading variance for bias.

Problem 3

3. Nonlinear Adaptive Filter with tanh Activation

Setup

Consider the adaptive system in Figure 3, where the linear combiner output $v(n) = \mathbf{W}_n^T X_n$ passes through a $\tanh(\cdot)$ nonlinearity before producing the filter output $y(n) = \tanh(v(n))$. The performance criterion is $J(\mathbf{W}_n) = E[\varepsilon^2(n)]$, estimated by $\hat{J}(\mathbf{W}_n) = \varepsilon^2(n)$, where $\varepsilon(n) = d(n) - y(n)$.

Derivation of the Update Rule

Applying the chain rule to the one-sample cost:

$$\nabla\hat{J}(\mathbf{W}_n) = \frac{\partial\varepsilon^2(n)}{\partial\mathbf{W}_n} = 2\varepsilon(n)\frac{\partial\varepsilon(n)}{\partial\mathbf{W}_n}$$

Since $\varepsilon(n) = d(n) - y(n)$, we have $\dfrac{\partial\varepsilon(n)}{\partial\mathbf{W}_n} = -\dfrac{\partial y(n)}{\partial\mathbf{W}_n}$. Using the chain rule through the nonlinearity:

$$\frac{\partial y(n)}{\partial \mathbf{W}_n} = \frac{dy}{dv}\cdot\frac{\partial v(n)}{\partial \mathbf{W}_n} = \bigl(1 - \tanh^2(v)\bigr)\,X_n = \bigl(1 - y^2(n)\bigr)\,X_n$$

Therefore:

$$\nabla\hat{J}(\mathbf{W}_n) = -2\varepsilon(n)\bigl(1 - y^2(n)\bigr)X_n$$

Steepest descent gives $\mathbf{W}_{n+1} = \mathbf{W}_n - \mu\nabla\hat{J}$:

$$\boxed{\mathbf{W}_{n+1} = \mathbf{W}_n + 2\mu\,\varepsilon(n)\bigl[1 - y^2(n)\bigr]X_n}$$
Key insight: The effective step size is modulated by $[1-y^2(n)] = \text{sech}^2(v(n))$, which is the derivative of $\tanh$. When the neuron is saturated ($|y(n)| \approx 1$), the effective step size shrinks toward zero — the algorithm naturally slows learning in the saturated regime.
