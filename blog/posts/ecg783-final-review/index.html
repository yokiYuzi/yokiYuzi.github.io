Problem 1

Adaptive Interference Canceller — Optimal Weight & $J_{\min}$

Setup

Consider a single-weight adaptive interference canceller. The primary input is

$$d(n) = s(n) + v(n),$$

and the reference input is

$$x(n) = v(n) + q(n).$$

The signals $s(n)$, $v(n)$, $q(n)$ are mutually uncorrelated, zero-mean, with variances $\sigma_s^2$, $\sigma_v^2$, $\sigma_q^2$ respectively. The filter has a single weight $w$, output $y(n) = wx(n)$, and error $e(n) = d(n) - y(n)$.

Part 1 — Performance Function $J(w)$

Substituting the signal model:

$$e(n) = s(n) + v(n) - w\bigl[v(n)+q(n)\bigr] = s(n) + (1-w)v(n) - wq(n).$$

Because $s$, $v$, $q$ are mutually uncorrelated:

$$\boxed{J(w) = \sigma_s^2 + (1-w)^2\sigma_v^2 + w^2\sigma_q^2.}$$

Expanding:

$$J(w) = \sigma_s^2 + \sigma_v^2 - 2w\sigma_v^2 + w^2(\sigma_v^2+\sigma_q^2).$$

Part 2 — Optimal Weight $w^*$

Setting $dJ/dw = 0$:

$$-2\sigma_v^2 + 2w^*(\sigma_v^2+\sigma_q^2) = 0.$$ $$\boxed{w^* = \frac{\sigma_v^2}{\sigma_v^2+\sigma_q^2}.}$$

Interpretation: the cleaner the reference (smaller $\sigma_q^2$), the closer $w^*$ is to 1, and the more completely the interference $v(n)$ is cancelled.

Part 3 — Minimum MSE $J_{\min}$

Substituting $w^*$ into $J(w)$ and simplifying:

$$\boxed{J_{\min} = \sigma_s^2 + \frac{\sigma_v^2\,\sigma_q^2}{\sigma_v^2+\sigma_q^2}.}$$

When $\sigma_q^2 = 0$ (perfect reference), $w^* = 1$ and $J_{\min} = \sigma_s^2$: the interference is completely removed and only the desired signal remains.

Part 4 — Orthogonality Condition $E[e(n)x(n)] = 0$

Compute directly at $w = w^*$:

$$E[e(n)x(n)] = E[d(n)x(n)] - w^*E[x^2(n)].$$

Since $s \perp v \perp q$:

$$E[d(n)x(n)] = E[(s+v)(v+q)] = \sigma_v^2, \qquad E[x^2(n)] = \sigma_v^2+\sigma_q^2.$$

Therefore:

$$E[e(n)x(n)] = \sigma_v^2 - \frac{\sigma_v^2}{\sigma_v^2+\sigma_q^2}\,(\sigma_v^2+\sigma_q^2) = 0. \qquad \boxed{\checkmark}$$

This is the Wiener–Hopf orthogonality principle: at the Wiener optimum, the error is orthogonal to the filter input.

Problem 2

2-Weight Performance Surface — Wiener Solution & Principal Coordinates

Setup

The MSE performance function for a 2-weight adaptive linear combiner is

$$J(W) = E[d^2(n)] - 2P^T W + W^T R W,$$

with

$$R = \begin{bmatrix}4&1\\1&2\end{bmatrix}, \quad P = \begin{bmatrix}5\\1\end{bmatrix}, \quad E[d^2(n)] = 10.$$

Part 1 — Optimal Weight Vector $W^*$

Solve the Wiener–Hopf equation $RW^* = P$:

$$R^{-1} = \frac{1}{\det R}\begin{bmatrix}2&-1\\-1&4\end{bmatrix} = \frac{1}{7}\begin{bmatrix}2&-1\\-1&4\end{bmatrix}.$$ $$\boxed{W^* = R^{-1}P = \frac{1}{7}\begin{bmatrix}2&-1\\-1&4\end{bmatrix}\begin{bmatrix}5\\1\end{bmatrix} = \frac{1}{7}\begin{bmatrix}9\\-1\end{bmatrix}.}$$

Part 2 — Minimum MSE $J_{\min}$

$$J_{\min} = E[d^2(n)] - P^T W^* = 10 - \frac{1}{7}(45-1) = 10 - \frac{44}{7} = \boxed{\frac{26}{7} \approx 3.714.}$$

Part 3 — Eigenvalues of $R$

$$\det(R-\lambda I) = (4-\lambda)(2-\lambda)-1 = \lambda^2-6\lambda+7 = 0.$$ $$\boxed{\lambda_{1,2} = 3 \pm \sqrt{2}; \qquad \lambda_{\max}=3+\sqrt{2}\approx4.414,\quad \lambda_{\min}=3-\sqrt{2}\approx1.586.}$$

Part 4 — Principal-Coordinate Representation

Let $V = W - W^*$ be the weight error vector. The performance function becomes $J = J_{\min} + V^T R V$. Diagonalising $R = Q\Lambda Q^T$ and substituting $V = QV'$ (principal coordinates):

$$\boxed{J(V') = \frac{26}{7} + (3+\sqrt{2})\,(v_1')^2 + (3-\sqrt{2})\,(v_2')^2.}$$

The performance surface is a paraboloid whose principal axes are the eigenvectors of $R$; the curvatures along those axes equal the eigenvalues.

Part 5 — Steepest-Descent Stability Range

The steepest-descent update is $W_{k+1} = W_k - \mu\nabla J$. The weight-error propagation factor along each eigen-direction is $1-2\mu\lambda_i$. For convergence $|1-2\mu\lambda_i|<1$ for all $i$, which requires

$$\boxed{0 < \mu < \frac{1}{\lambda_{\max}} = \frac{1}{3+\sqrt{2}} \approx 0.2265.}$$

Part 6 — Convergence Check at $\mu = 0.2$

Since $0.2 < 0.2265$ the algorithm converges. However, the factor along the maximum-eigenvalue direction is

$$1 - 2(0.2)(3+\sqrt{2}) \approx -0.766 < 0,$$

so convergence along that direction is oscillatory (sign alternates each iteration, amplitude decays).

Problem 3

Finite-Difference Gradient Estimation — Variance & Perturbation Trade-off

Setup

In the translated coordinate $v = w - w^*$ the performance function is $J(v) = J_{\min} + \lambda v^2$. The true gradient is $g(v) = dJ/dv = 2\lambda v$. Since $J$ cannot be computed exactly, it is estimated from $N$ samples as $\hat J(v) = \frac{1}{N}\sum e^2(n)$, and the gradient is estimated by the central difference

$$\widehat{g}(v) = \frac{\hat J(v+s) - \hat J(v-s)}{2s}.$$

Assume $\operatorname{var}[\hat J] \approx kJ_{\min}^2/N$ near the optimum, independent at $\pm s$.

Part 1 — Ideal Central-Difference Result

$$J(v+s)-J(v-s) = \lambda[(v+s)^2-(v-s)^2] = 4\lambda vs,$$ $$\boxed{g(v) = \frac{4\lambda vs}{2s} = 2\lambda v.} \quad \text{(exact gradient)}$$

Part 2 — Gradient Estimate Variance

By independence of the two estimates and linearity of variance:

$$\operatorname{var}[\widehat{g}(v)] = \frac{1}{4s^2}\Bigl[\operatorname{var}[\hat J(v+s)]+\operatorname{var}[\hat J(v-s)]\Bigr] = \frac{1}{4s^2}\cdot\frac{2kJ_{\min}^2}{N}.$$ $$\boxed{\operatorname{var}[\widehat{g}(v)] = \frac{kJ_{\min}^2}{2Ns^2}.}$$

Key insight: variance grows as $s \to 0$, so a smaller perturbation does not automatically improve the gradient estimate.

Part 3 — Normalised Performance Perturbation

The excess MSE caused by operating at $w^* + s$ rather than $w^*$ is $\gamma = \lambda s^2$. Normalised:

$$\boxed{P = \frac{\gamma}{J_{\min}} = \frac{\lambda s^2}{J_{\min}}.}$$

Part 4 — Numerical Example

Given $\lambda=4$, $J_{\min}=1$, $k=2$, $N=1000$, target $P=0.01$:

$$s^2 = \frac{P\,J_{\min}}{\lambda} = \frac{0.01}{4} = 0.0025 \implies \boxed{s = 0.05.}$$ $$\operatorname{var}[\widehat{g}] = \frac{2\cdot 1^2}{2\cdot 1000\cdot 0.0025} = \frac{2}{5} = \boxed{0.4.}$$
Trade-off summary: A larger $s$ reduces gradient variance but increases performance perturbation $P$. A smaller $s$ reduces $P$ but inflates variance as $1/s^2$. The perturbation tolerance $P$ is therefore the primary constraint for choosing $s$.
Problem 4

LMS Stability, Misadjustment & LMS/Newton Comparison

Setup

A 4-tap adaptive FIR filter uses LMS: $W_{n+1} = W_n + 2\mu e(n)X_n$. The input correlation matrix $R$ has eigenvalues $\lambda_0=6$, $\lambda_1=3$, $\lambda_2=1$, $\lambda_3=0.5$. Minimum MSE $J_{\min} = 0.2$.

Part 1 — LMS Stability Range

Convergence requires $|1-2\mu\lambda_i|<1$ for all $i$, so $\mu<1/\lambda_{\max}$:

$$\boxed{0 < \mu < \frac{1}{6} \approx 0.1667.}$$

Part 2 — Misadjustment at $\mu = 0.02$

The approximate LMS misadjustment formula:

$$M \approx \mu\,\operatorname{tr}(R) = \mu\sum_{i}\lambda_i = 0.02\times(6+3+1+0.5).$$ $$\boxed{M_{\mathrm{LMS}} \approx 0.02 \times 9.5 = 0.19.}$$

Part 3 — Excess MSE

$$J_{\mathrm{excess}} = M\,J_{\min} = 0.19\times 0.2 = \boxed{0.038.}$$

Part 4 — Slowest MSE Time Constant

The MSE time constant along eigen-direction $i$ is $\tau_{\mathrm{MSE},i} \approx 1/(4\mu\lambda_i)$. The slowest mode corresponds to $\lambda_{\min} = 0.5$:

$$\tau_{\mathrm{MSE,slow}} = \frac{1}{4\times 0.02\times 0.5} = \boxed{25\text{ samples.}}$$

Part 5 — LMS/Newton ($\mu_N = 0.02$)

LMS/Newton replaces the gradient by $R^{-1}$ times the gradient: $W_{n+1} = W_n + 2\mu_N R^{-1}e(n)X_n$. This normalises all eigen-directions to the same effective step size.

MSE time constant (all modes identical):

$$\tau_{\mathrm{MSE,Newton}} = \frac{1}{4\mu_N} = \frac{1}{4\times0.02} = \boxed{12.5\text{ samples.}}$$

Misadjustment (approximate, $L+1=4$ weights):

$$M_{\mathrm{Newton}} \approx (L+1)\,\mu_N = 4\times 0.02 = \boxed{0.08.}$$

More precise form accounting for $1-\mu_N$ denominator:

$$M_{\mathrm{Newton}} = \frac{(L+1)\mu_N}{1-\mu_N} = \frac{0.08}{0.98} \approx 0.0816.$$

Excess MSE (Newton):

$$J_{\mathrm{excess,Newton}} \approx 0.08\times 0.2 = \boxed{0.016.}$$
Algorithm$M$$J_{\text{excess}}$$\tau_{\text{slow}}$
LMS ($\mu=0.02$)0.190.03825 samples
LMS/Newton ($\mu_N=0.02$)0.080.01612.5 samples

LMS/Newton achieves lower misadjustment and faster convergence because $R^{-1}$ equalises the curvature across all weight-space directions.

Problem 5

$z$-Domain Performance Function, IIR LMS, SER & Stability Triangle

Setup

Adaptive IIR filter: $H(z) = (a_0 + a_1 z^{-1})\,/\,(1 - b_1 z^{-1})$, time-domain form $y(n) = a_0 x(n) + a_1 x(n-1) + b_1 y(n-1)$, error $e(n)=d(n)-y(n)$. Input $x(n)$ is white noise with variance $\sigma_x^2$. Unknown plant: $H_p(z)=1+\tfrac{1}{2}z^{-1}$, so $d(n)=x(n)+\tfrac{1}{2}x(n-1)$.

Part 1 — $z$-Domain Performance Function

The general MSE is $J = E[e^2(n)]$. In the $z$-domain, for stationary inputs:

$$J = \phi_{dd}(0) + \mathcal{Z}^{-1}\!\left\{H(z)H(z^{-1})\,\Phi_{xx}(z) - 2\,\Phi_{dx}(z)\,H(z)\right\}_{n=0},$$

where $\Phi_{xx}(z)$ is the input power spectral density and $\Phi_{dx}(z)$ is the cross-PSD.

For white noise input $\Phi_{xx}(z)=\sigma_x^2$, and $\Phi_{dx}(z)=H_p(z^{-1})\Phi_{xx}(z)=\sigma_x^2(1+\tfrac{1}{2}z)$, giving

$$\boxed{J = \phi_{dd}(0) + \mathcal{Z}^{-1}\!\left\{\sigma_x^2 H(z)H(z^{-1}) - 2\sigma_x^2\!\left(1+\tfrac{1}{2}z\right)\!H(z)\right\}_{n=0}.}$$

Part 2 — IIR Sensitivity Recursions

Define sensitivity signals as partial derivatives of $y(n)$ with respect to each parameter. Differentiating the recursion $y(n)=a_0 x(n)+a_1 x(n-1)+b_1 y(n-1)$:

$$\boxed{\alpha_0(n) = \frac{\partial y(n)}{\partial a_0} = x(n) + b_1\,\alpha_0(n-1),}$$ $$\boxed{\alpha_1(n) = \frac{\partial y(n)}{\partial a_1} = x(n-1) + b_1\,\alpha_1(n-1),}$$ $$\boxed{\beta_1(n) = \frac{\partial y(n)}{\partial b_1} = y(n-1) + b_1\,\beta_1(n-1).}$$

Each sensitivity signal satisfies the same IIR recursion driven by the corresponding "input" ($x(n)$, $x(n-1)$, or $y(n-1)$), with the same pole $b_1$.

Part 3 — IIR LMS Update

Define $W_n = [a_0,a_1,b_1]^T$ and the sensitivity vector $\Psi_n = [\alpha_0(n),\alpha_1(n),\beta_1(n)]^T$.

The instantaneous gradient is $\nabla_W\hat J(n) = -2e(n)\Psi_n$, so the LMS update is

$$\boxed{W_{n+1} = W_n + 2\mu\,e(n)\,\Psi_n,}$$

i.e.:

$$\begin{bmatrix}a_0(n+1)\\a_1(n+1)\\b_1(n+1)\end{bmatrix} = \begin{bmatrix}a_0(n)\\a_1(n)\\b_1(n)\end{bmatrix} + 2\mu\,e(n)\begin{bmatrix}\alpha_0(n)\\\alpha_1(n)\\\beta_1(n)\end{bmatrix}.$$

Part 4 — SER Inverse-Matrix Recursion

The SER (Stochastic-Estimation RLS) approach maintains an exponentially-weighted correlation matrix $C_n = \alpha C_{n-1} + U_n U_n^T$ and updates its inverse via the matrix inversion lemma. Define $S_n = C_{n-1}^{-1}U_n$. Then:

$$\boxed{C_n^{-1} = \frac{1}{\alpha}\left[C_{n-1}^{-1} - \frac{S_n S_n^T}{\alpha + U_n^T S_n}\right], \qquad S_n = C_{n-1}^{-1}U_n.}$$

For IIR adaptive filters, $U_n$ is the sensitivity vector $\Psi_n$ (or a regressor built from both $X_n$ and $y(n-1)$ depending on formulation). The resulting LMS/Newton-SER update becomes

$$W_{n+1} = W_n + \frac{2\mu}{1-\alpha}\,C_n^{-1}\,e(n)\,U_n.$$

Part 5 — Second-Order IIR Stability Triangle

For the system $H(z)=a_0\,/\,(1-b_1 z^{-1}-b_2 z^{-2})$, stability requires all poles inside the unit circle. The necessary and sufficient conditions are:

$$\boxed{-1 < b_2 < 1,\qquad b_2 > b_1 - 1,\qquad b_2 > -b_1 - 1.}$$

Equivalently: $-1 < b_2 < 1 - |b_1|$.
